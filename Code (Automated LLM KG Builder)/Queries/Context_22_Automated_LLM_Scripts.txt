How many dates did Beyonce's "The Mrs. Carter Show" entail?
MATCH (b:Celebrity {name: 'Beyoncé'})
MATCH (a:Album {title: 'Beyoncé', releaseDate: '2013-12-13'})
MATCH (b)<-[:OWNS]-(a)
MATCH (a)<-[:BELONGS_TO]-(c:Chart {name: 'Billboard 200 chart'})
MATCH (a)<-[:TOP_SELLING]-(c)
MATCH (a)<-[:DEBUTS_AT_NUMBER_ONE]-(c)
